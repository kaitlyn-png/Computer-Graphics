# CSE 160: Computer Graphics Repository
## Assignment 0: Vector Library
- Creating object oridented graphics projects in Javascript

## Assignment 1: Painting
- Creating a web application to draw simple geometric shapes made out of points and triangles

## Assignment 2: Blocky World
- To transform (translation, rotation, scaling, etc) simple 2D geometric shapes and 3D objects using matrices. 
- Create a simple 3D animal out simple geometric objects, like cubes, spheres, and cylinders.

## Assignment 3: Creating a Virtual World
- To create a virtual world using textured cubes and explore it using a perspective camera.

## Assignment 4: Lighting
- To have a light that illuminates your objects.

## Assignment 5: Three.js
- Review all the topics covered in this course.
- Learn to use a a high-level 3D library.
- Practice learning new skills directly from web resources
